# FlapDash - Quick Start

A fun and addictive Flappy Bird-style game built with Python and Pygame!

## 📁 Project Structure

```
FlapDash/
├── src/
│   └── main.py                  # Main game file
├── assets/
│   ├── images/                  # Game graphics go here
│   └── sounds/                  # Audio files go here
├── data/
│   └── score.text               # High score (auto-generated)
└── docs/
    └── README.md                # Full documentation
```

## 🚀 Quick Start

1. **Place your assets** in the appropriate directories:
   - All images → `assets/images/`
   - All sounds → `assets/sounds/`

2. **Run the game** from the src directory:
   ```bash
   cd src
   python main.py
   ```
